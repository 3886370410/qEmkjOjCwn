# Python图书借阅推荐系统

## 核心功能
- 图书借阅
- 图书信息查阅（最新、最热、推荐）
- 图书搜索
- 图书分类
- 图书详情
- 图书信息管理
- 分类管理
- 评论管理
- 标签管理

## 适用场景
- 图书馆数字化管理系统
- 学校图书馆资源利用优化
- 个人图书管理推荐系统

## 技术特点
- 使用 Django 框架构建后端，确保系统稳定可靠
- 基于 MySQL 数据库存储图书信息，提供高效的数据查询
- 采用 Vue.js 前端技术，提供友好的用户界面和交互体验
